Materi - Aziizah Nurul Amini - Arsitektur dan Jaringan Komputer.

1. ROUTER itu apasi..??

Router itu sebuah perangkat yang memiliki tugas utama untuk mengirimkan paket data dari jaringan internet menuju perangkat lain. Sederhananya, router bekerja untuk mengelola arus lalu lintas jaringan dengan meneruskan paket data ke berbagai alamat IP tujuan. Kegunaan utama router adalah untuk menerima, menganalisis, dan meneruskan paket data antar jaringan.

2. FUNGSI nya..??

Router memberikan berbagai fungsi yang berguna untuk kehidupan teknologi masa kini. Diantara nya yaitu Mengirimkan informasi, Menghubungkan jaringan, Menyaring paket data, Menghubungkan jaringan ke DSL, Membagikan file.

3. CARA KERJA nya gimana..??

Router bekerja dengan memastikan paket data atau internet sampai ke perangkat-perangkat yang terhubung secara efektif. Dalam melakukan tugasnya, router bekerja dengan bantuan table routing untuk menganalisis dan mengirimkan paket data atau internet.

4. JENIS ROUTER itu apa aja..??

Router terdiri dari berbagai jenis yang dibedakan berdasarkan dua hal, yaitu Mekanisme dan Pengaplikasiannya.

Berdasarkan Mekanismenya.

1. Router Statis, Router jenis ini merupakan router dengan tabel routing tetap. Router jenis ini dikonfigurasi secara manual oleh administrator jaringan. Router statis cocok untuk kebutuhan jaringan Internet kecil.

2. Router Dinamis, Berbeda dengan router statis, router dinamis mempunyai tabel routing yang berubah-ubah. Router secara otomatis mempelajari aliran optimal untuk merutekan paket data sesuai dengan instruksi administrator. Jenis router dinamis ini terutama digunakan untuk kebutuhan Internet bervolume tinggi.

Berdasarkan Pengaplikasiannya.

1. Router Aplikasi, yaitu aplikasi yang bisa di instal pada sistem operasi komputer sehingga sistem operasi komputer dapat bekerja layaknya router, Misalnya aplikasi WinGate, WinProxy, Winroute, dan SpyGate.

2. Router Hardware, yaitu hardware yang berkemampuan seperti router sehingga dapat membagi IP Address, Router ini dapat digunakan untuk membagi jaringan internet dalam satu wilayah, Misalnya bila router ini adalah access point, wilayah yang mendapat IP Address dan koneksi internet disebut HotSpot Area.

3. Router PC, yaitu sebuah komputer yang dimdifikasi agar berfungsi layaknya router, Contoh sistem operasi populer untuk router PC adalah Mikrotik.

Demikian pembahasan mengenai router, yaitu perangkat jaringan yang digunakan untuk mengirimkan paket data dari jaringan internet ke tujuannya melalui proses routing. Router memiliki beragam fungsi yang membuat pennyebaran Internet lebih mudah. Ada berbagai jenis berdasarkan mekanisme dan pengaplikasianny.
